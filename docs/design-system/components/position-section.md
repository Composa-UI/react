# PositionSection

PositionSection is the right-panel transform section for alignment, X/Y position, constraints, rotation, and flip controls.

## Anatomy

| Part | Composes |
| --- | --- |
| Header | 40px section heading row |
| Field set | Small label over one dense 24px control row |
| Alignment | Two icon groups plus more actions |
| Position | X/Y numeric inputs plus constraints |
| Rotation | Rotation numeric input plus rotate/flip action group |

## Guidelines

- Use the newer vertical field-set layout when labels need to describe grouped controls.
- Keep each control row 24px tall so the full field set lands at 48px.
- Keep the section width fluid, but verify at the 240px right-panel width.
- Use official Material Symbols names through the Composa icon runtime; do not draw local placeholder icons.
