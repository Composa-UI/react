# Sizing & Shape

Sizing covers control heights, corner radii, and icon dimensions. These tokens come from the Sizing collection plus a small set of hand-authored component values.

export const Row = ({ token, value, note }) => (
  <div style={{ display: "flex", alignItems: "center", gap: 16, padding: "8px 0", borderBottom: "1px solid var(--composa-color-border-translucent)" }}>
    <div style={{ minWidth: 230, flexShrink: 0, fontFamily: "monospace", fontSize: 12 }}>{token}</div>
    <div style={{ minWidth: 70, flexShrink: 0, fontFamily: "monospace", fontSize: 12, opacity: 0.7 }}>{value}</div>
    {note ? <div style={{ fontSize: 12, opacity: 0.7 }}>{note}</div> : null}
  </div>
);

## Control heights

The input height is the standard dense-control unit. Use it for the smallest buttons, dropdowns, inputs, tabs, segmented items, and icon controls instead of mixing 30px and 32px control heights. The heading height sizes toolbars and section headers.

<div style={{ margin: "12px 0 16px" }}>
  <Row token="--composa-height-input" value="24px" note="dense controls" />
  <Row token="--composa-height-heading" value="40px" note="toolbar / heading row" />
</div>

A 24px control rendered at the input height:

<div style={{ display: "inline-flex", alignItems: "center", height: "var(--composa-height-input)", padding: "0 var(--composa-space-2)", background: "var(--composa-color-bg-secondary)", border: "1px solid var(--composa-color-border)", borderRadius: "var(--composa-radius-medium)", fontFamily: "var(--composa-font-family)", fontSize: "var(--composa-body-medium-size)", marginBottom: 28 }}>
  height-input · 24px
</div>

## Row heights

Rows should consolidate around two heights: heading rows and item rows. Heading rows are 40px tall so a 24px control can sit centered with 8px total vertical breathing room. List and item rows are 32px tall.

<div style={{ margin: "12px 0 28px" }}>
  <Row token="heading row" value="40px" note="section headers, toolbars, panel headers" />
  <Row token="item row" value="32px" note="list rows, layer rows, repeated items" />
  <Row token="dense control" value="24px" note="buttons, dropdowns, inputs, tabs, icon controls" />
</div>

## Radius

The radius scale runs none, small, medium, large, full. Controls use medium. Menus and dialogs use large. Pills and avatars use full. The large value (13px) is hand-authored to preserve token parity with the menu corner.

export const RadiusSwatch = ({ token, value, radius }) => (
  <div style={{ display: "flex", alignItems: "center", gap: 16, padding: "8px 0", borderBottom: "1px solid var(--composa-color-border-translucent)" }}>
    <div style={{ width: 48, height: 48, flexShrink: 0, background: "var(--composa-color-bg-secondary)", border: "1px solid var(--composa-color-border)", borderRadius: `var(${radius})` }} />
    <div style={{ minWidth: 200, fontFamily: "monospace", fontSize: 12 }}>{token}</div>
    <div style={{ fontFamily: "monospace", fontSize: 12, opacity: 0.7 }}>{value}</div>
  </div>
);

### Nesting

Radii nest concentrically: an inner element uses `radius-medium` (5px); its parent container uses `radius-large` (13px). With the default 8px gap between them the curves stay parallel — because **5 + 8 = 13**. The outer box below is `radius-large` with `space-2` padding around a `radius-medium` inner box.

<div style={{ display: "inline-flex", margin: "12px 0 28px", padding: "var(--composa-space-2)", background: "var(--composa-color-bg-secondary)", border: "1px solid var(--composa-color-border)", borderRadius: "var(--composa-radius-large)" }}>
  <div style={{ width: 132, height: 72, background: "var(--composa-color-bg-brand-tertiary, rgba(255,92,22,0.12))", border: "1px solid var(--composa-color-bg-brand)", borderRadius: "var(--composa-radius-medium)" }} />
</div>

## Icons

Icons ship at two sizes. The frame is 24px. The glyph inside it is 16px. Both are exposed as control-size tokens.

export const IconBox = ({ size, label }) => (
  <div style={{ display: "flex", flexDirection: "column", alignItems: "center", gap: 8 }}>
    <div style={{ width: `var(${size})`, height: `var(${size})`, background: "var(--composa-color-bg-secondary)", border: "1px solid var(--composa-color-border)", borderRadius: "var(--composa-radius-small)" }} />
    <div style={{ fontFamily: "monospace", fontSize: 11, opacity: 0.7 }}>{label}</div>
  </div>
);

<div style={{ display: "flex", gap: 24, alignItems: "flex-end", margin: "12px 0 16px" }}>
  <IconBox size="--composa-control-size-icon" label="icon · 24px" />
  <IconBox size="--composa-control-size-icon-glyph" label="glyph · 16px" />
</div>

<div style={{ margin: "0 0 16px" }}>
  <Row token="--composa-control-size-icon" value="24px" note="icon frame" />
  <Row token="--composa-control-size-icon-glyph" value="16px" note="glyph inside the frame" />
</div>
