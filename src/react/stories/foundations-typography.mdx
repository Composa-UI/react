# Typography

The product UI uses Inter. The family token is `--composa-font-family`, which falls back through the system sans-serif stack.

```
Inter, ui-sans-serif, system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif
```

The scale has two ladders: headings (display, large, medium, small) and body (large, medium, small). Body roles carry a regular weight of 450 and a strong weight of 550. Every role carries an explicit line-height and letter-spacing token. The samples below render at the live token values.

export const Sample = ({ label, family, size, line, weight, letter, text }) => (
  <div style={{ padding: "14px 0", borderBottom: "1px solid var(--composa-color-border-translucent)" }}>
    <div
      style={{
        fontFamily: `var(${family})`,
        fontSize: `var(${size})`,
        lineHeight: `var(${line})`,
        fontWeight: `var(${weight})`,
        letterSpacing: letter ? `var(${letter})` : "normal",
        margin: 0,
      }}
    >
      {text}
    </div>
    <div style={{ marginTop: 6, fontFamily: "monospace", fontSize: 11, opacity: 0.7 }}>
      {label}
    </div>
  </div>
);

## Headings

Headings step from a 48px display down to a 13px small. Weight is 450 at display and 550 for large, medium, and small. Letter-spacing tightens as size grows.

## Type scale

Composa carries the full type scale as tokens. The named roles are display, large, medium, and small for headings, and large, medium, and small for body, each body role with a strong sibling. Body large is 13/22 and body medium is 11/16.
