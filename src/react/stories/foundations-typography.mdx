# Typography

The product UI uses Inter. The family token is `--composa-font-family`, which falls back through the system sans-serif stack.

```
Inter, ui-sans-serif, system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif
```

The scale has two ladders: headings (display, large, medium, small) and body (large, medium, small). Body roles carry a regular weight of 450 and a strong weight of 550. Every role carries an explicit line-height and letter-spacing token. The samples below render at the live token values.

export const Row = ({ token, desc, children }) => (
  <div style={{ display: "flex", gap: 32, alignItems: "flex-start", padding: "18px 0", borderBottom: "1px solid var(--composa-color-border-translucent)" }}>
    <div style={{ width: 190, flexShrink: 0 }}>
      <div style={{ fontFamily: "monospace", fontSize: 12, color: "var(--composa-color-text)" }}>{token}</div>
      {desc ? <div style={{ fontSize: 12, opacity: 0.7, marginTop: 6, lineHeight: 1.45 }}>{desc}</div> : null}
    </div>
    <div style={{ flex: 1, minWidth: 0, display: "flex", gap: 16, flexWrap: "wrap" }}>{children}</div>
  </div>
);

export const Spec = ({ family, size, line, weight, letter, text, meta }) => (
  <div style={{ flex: "1 1 0", minWidth: 240, padding: "16px 20px", border: "1px solid var(--composa-color-border-translucent)", borderRadius: 8, background: "var(--composa-color-bg-secondary)" }}>
    <div style={{ fontFamily: `var(${family || "--composa-font-family"})`, fontSize: `var(${size})`, lineHeight: `var(${line})`, fontWeight: `var(${weight})`, letterSpacing: letter ? `var(${letter})` : "normal", margin: 0, color: "var(--composa-color-text)" }}>{text}</div>
    {meta ? <div style={{ marginTop: 10, fontFamily: "monospace", fontSize: 11, opacity: 0.6 }}>{meta}</div> : null}
  </div>
);

## Headings

Headings step from a 48px display down to a 13px small. Weight is 450 at display and 550 for large, medium, and small. Letter-spacing tightens as size grows.

## Type scale

Composa carries the full type scale as tokens. The named roles are display, large, medium, and small for headings, and large, medium, and small for body, each body role with a strong sibling. Body large is 13/22 and body medium is 11/16.
